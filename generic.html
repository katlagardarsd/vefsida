Verkefni 1:
Vefsíðugerð

Fyrsta verkefni annarinnar var að búa til vel skipulagða vefsíðu sem heldur utan um öll verkefnin sem verða unnin í áfanganum VÉL608G en hér má lesa nánar um áfangann.

Þar sem ég hafði ekki mikla þekkingu á vefsíðu gerð fyrir þetta verkefni byrjaði ég á því að horfa á myndbönd HTML myndband1 og HTML myndband2 á Youtube. Mælt var með forritinu Brackets svo að ég hlóð því niður. Því næst fór ég inn á vefsíðuna html5up.net til að velja mér sniðmát af vefsíðu og ég ákvað að nota sniðmátið Massively.
Mér fannst grái liturinn á sniðmátinu flottur en ég ákvað að prófa mig áfram og breyta bakgrunninum. Þá fór ég inn í möppuna Assets og þaðan inn í möppuna CSS og þar er hægt að breyta m.a. bakgrunninum, leturstærð, leturgerð o.fl. Hér fyrir neðan má sjá kóðann þar sem ég breytti bakgrunninum. Mig langaði að bæta inn takka svo hægt sé að fara til baka neðst á þessa síðu og það gerði ég með því að setja inn kóðann:

Þegar að vefsíðan var tilbúin var næsta skref að hlaða henni upp í Github. Þar sem ég var ekki með aðgang að Github byrjaði ég á því að búa mér til aðgang. Þar á eftir horfði ég á GITHUB myndband1 á Youtube og fór eftir þeim leiðbeiningum sem komu fram í myndbandinu en ég mun fara yfir það sem ég gerði í nokkrum skrefum.

  1. Það fyrsta sem ég gerði var að fara inn á Github aðganginn minn og búa til New repository. Þann hnapp má finna undir plúsinum í hægra horninu á Github síðunni.
  2. Næst fyllum við inn í gluggann Repository name og pössum að það sé hakað í Public hnappinn þar sem við viljum að aðrir geti séð vefsíðuna. Þegar það er komið er ýtt á hnappinn Create repository.
  3. Næst hægri-smellum við á möppuna sem vefsíðan okkar er geymd í og veljum New terminal at folder og þá ætti að opnast terminal gluggi. Þá förum við aftur inn á Github og afritum textann á síðunni sem opnaðist inn á terminal gluggann.

    Þegar búið er að afrita kóðann inn á terminal gluggann og ýta á enter þarf maður að skrá notendanafn og lykilorð (Token) inn. Ég lenti í smá vandræðum þar því ég var ekki búin að búa mér til Token en ég gerði það með því að fara eftir leiðbeiningum sem ég fann á þessari síðu.
  4. Nú ættu allar skrár að vera komnar inn á Github síðuna og ætti síðan þá að líta svona út.

    Þá förum við í settings í hægra horninu og þar förum við í GitHub pages. Þar breytum við None í Main og vistum breytingarnar og þá ætti síðan að líta svona út:
    Nú er hægt að nota þennan link sem birtist til þess að deila síðunni með öðrum.
Þurfti svo að breyta síðunni og halda áfram að vinna í henni svo sýna hvernig ég gerði það í terminal.
